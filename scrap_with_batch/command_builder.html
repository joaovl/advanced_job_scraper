Options

Search Options
-k, --keywords
Job title or search terms
-g, --geo-id
LinkedIn geoId for precise location filtering (recommended)
-l, --location
Text location (use --geo-id instead for accuracy)
Filters
-t, --time-range
Only jobs posted within this time frame
-n, --max-jobs
Maximum jobs per search (0 = scrape all available)
--max-age
Local filter: only keep jobs posted within this time
Output Options
-o, --output
Custom output filename
-w, --workers
Parallel threads for descriptions (1-5, default: 2)
